# TanStack Start + shadcn/ui

This is a template for a new TanStack Start project with React, TypeScript, and shadcn/ui.

## Project structure

- `src/components/`: React components and UI primitives
- `src/helpers/`: feature-specific helpers (non-UI composition logic)
- `src/lib/`: service clients and infrastructure wiring (auth, email, database)
- `src/routes/`: TanStack routes and server handlers
- `src/utils/`: reusable utilities (class names, env helpers, server helpers)

## Coding style

- Formatting: Prettier (see `prettier.config.js`)
- Linting: ESLint (see `eslint.config.js`)
- Documentation: JSDoc for non-React modules
